# 城鎮

這是遊戲本身的主畫面 剛登入遊戲 會進到這裡
僅有在這裡才能進行以下事情 !

## 有diamond.png 可以執行領鑽石分支流程
領鑽石流程:
1. 檢查diamond.png
2. 點擊diamond.png
3. 等待領鑽石畫面
4. 如果有free.png 就點擊
5. 如果沒有free.png 就點擊quit.png
6. 點擊後,流程結束 會回到城鎮的圖樣

## 有 door.png 可以進入大廳


# 大廳

僅有在這裡才能進行以下事情 ! 這是 去關卡(select_stage.png) 或地下城(dungeon.png)的地方,也可以在這裡領取體力(bread.png), 也可以從這裡回去城鎮(goback_town.png)

## 領取體力(bread.png)流程:
1. 檢查bread.png
2. 點擊bread.png
3. 等待領體力畫面
4. 如果有collect.png 就點擊collect.png
5. 如果沒有collect.png 就點擊quit.png
6. 點擊後,流程結束 會回到大廳
